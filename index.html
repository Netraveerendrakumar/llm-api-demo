⚡ What is n8n?

An open-source workflow automation tool. You connect nodes visually — each node does one job like fetching data, calling an API, or sending a message.

🤖 What is an LLM API?

A POST request to an AI endpoint like /v1/messages. You send a messages[] array and get back an AI-generated text response.

🔗 How they connect

n8n has a built-in AI Agent node that calls LLM APIs for you. Under the hood it sends the same JSON payload you see here.
